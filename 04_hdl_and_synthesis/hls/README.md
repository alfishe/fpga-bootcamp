# High-Level Synthesis (HLS)

HLS raises the abstraction level from RTL to C/C++, OpenCL, or MATLAB, automatically generating hardware via scheduling, binding, and resource allocation. This section covers HLS from the HDL designer's perspective — when to use it, what it produces, and how to integrate generated RTL into a larger design.
