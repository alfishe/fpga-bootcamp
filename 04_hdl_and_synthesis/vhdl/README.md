# VHDL

VHDL (VHSIC Hardware Description Language) is a strongly-typed, IEEE-standardized HDL. VHDL-2008 is the baseline for modern FPGA synthesis, offering robust type systems, hierarchical design via entities/architectures, and extensive standard libraries.
