# 06-F — Transceivers & SerDes

Multi-gigabit transceivers are the workhorses behind PCIe, 10G+ Ethernet, SATA, DisplayPort, and high-speed ADC/DAC links. This folder covers the PMA+PCS architecture and how to configure these blocks correctly.
