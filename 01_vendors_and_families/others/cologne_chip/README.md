# Cologne Chip GateMate — German Open-Source FPGA

A small German FPGA with open-source toolchain support (Yosys + nextpnr). Only one device exists but represents European sovereignty in logic silicon.

---

| Feature | GateMate CCGM1A1 |
|---|---|
| **CPE (logic cells)** | 20,480 |
| **Block RAM** | 128 Kb |
| **PLL** | 2 |
| **Max IOs** | 80 |
| **Package** | QFN64 |
| **Price** | ~$15 |
| **Toolchain** | Yosys + nextpnr (open) or Cologne Chip IDE (proprietary) |

Not competitive with bigger vendors on density/price, but significant in open-source and geopolitical context.
