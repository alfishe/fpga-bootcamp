# 01-D — Gowin Semiconductor

The ultra-low-cost challenger from China — Tang Nano boards start under $10. Gowin's LittleBee and Arora families cover the low-to-mid range, with a free (but proprietary) IDE and growing open-source toolchain support.

## Family Directories

| Directory | Coverage |
|---|---|
| [littlebee/](littlebee/README.md) | **LittleBee (55nm)** — GW1N-1 through GW1N-9C, GW1NR (integrated flash+PSRAM), GW1NSR (hard PicoRV32 SoC). $1.50–$10 chips, Tang Nano boards from $6. Project Apicula open-source flow |
| [arora/](arora/README.md) | **Arora GW2A (55nm) & Arora-V GW5A (22nm)** — DDR2/DDR3/LPDDR4 hard controllers, 2.5–6.25 Gbps SERDES, PCIe Gen2 ×4 (GW5A-25), Tang Primer boards ($40–50) |

## Quick Reference

| Family | Key Devices | Notes |
|---|---|---|
| **LittleBee (GW1N/R/NR/Z/NS)** | GW1N-1/4/9, GW1NSR (SoC with PicoRV32) | Low-cost, 55nm, Tang Nano boards. Open-source nextpnr support (project Apicula/Nano) |
| Arora (GW2A) | GW2A-18/55 | Mid-range, more LUTs and DSP, Tang Primer |
| Arora-V (GW5A) | GW5A-25 | Newer Arora, 22nm, more performance, PCIe |
| GW5AT | GW5AT-138 | High-end Gowin, SerDes, large capacity |
