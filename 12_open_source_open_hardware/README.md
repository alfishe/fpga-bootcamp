# 12 — Open Source & Open Hardware

The sprawling ecosystem of community-driven FPGA projects, platforms, and initiatives. From retro computing emulation (MiSTer, MiST) to open development boards, open-source CPU and peripheral cores, memory controllers, video scalers (OSSC), networking stacks, GPU projects, and industry-shaping organizations.

**This is the inspiration section — where you find what's possible and download something that already works.**
